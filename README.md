# Capstone---DB
This is a simple program to be used by a bookstore clerk. The clerk can:

> ○ add new books to the database

> ○ update book information

> ○ delete books from the database

> ○ search the database to find a specific book.

The prompted menu:

1. Enter book
2. Update book 
3. Delete book 
4. Search books 
0. Exit
